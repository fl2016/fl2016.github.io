进程资源

#include <sys/resource.h>
int getrusage(int who,struct rusage *res_usage);

　　who参数指定了需查询使用信息的资源。其取值为以下几个值中的一个。
　　RUSAGE_SELF 返回进程相关的信息
　　RUSAGE_CHILDREN 返回调用进程的所有被终止和处于等待状态的子进程的相关消息
　　RUSAGE_THREAD 返回线程相关的信息，这个是linux特有的。
　　res_usage参数是一个指向rusage结构的指针。

进程资源限制

　　每个进程都有一组资源限值，使用shell的内置命令ulimit可以设置shell的资源限制，shell创建用来执行用户命令的进程会继承这些限制。

#include <sys/resource.h>
int getrlimit(int resource,struct rlimit *rlim);
int setrlimit(int resource,struct rlimit *rlim);

　　resource参数标识出了需要读取或者修改的资源限制，rlim参数用来返回限制值，它是一个指向包含两个字段结构的指针。

struct rlimit{
   rlim_t rlim_cur;//soft limit（actual process limit）
   rlim_t rlim_max;//hard limit(ceiling for rlim_cur)
   }

　　特权进程(CAP_SYS_RESOURCE)能够增大或者缩小硬限制，非特权进程只能缩小硬限制。
　　通过fork()创建的子进程会继承父进程的限制，并且在exec()调用之间不得不保持。 　　

daemon

　　daemon是一种具备以下特征的进程：

  • 生命周期很长，通常是系统启动时被创建直到系统被关闭。
  • 他在后台运行且不拥有控制终端。

　　daemon是用来执行特殊任务的：
　　1.cron 一个在规定时间内执行命令的daemon.
　　2.sshd 安全shell daemon,允许远程主机使用一个安全的通信协议登录系统。
　　3.httpd:HTTP服务器daemon。
　　4.inetd:Internet超级服务器daemon，它监听从指定的TCP/IP口进入的网络连接并启动相应的服务器程序来处理这些连接。
